ion-intellij-plugin
===================

This is an IntelliJ IDEA plugin to support the [Amazon Ion Language Specification](http://amzn.github.io/ion-docs/spec.html).

If you are unable to install the plugin from JetBrains Marketplace, you can download the plugin from the [latest release](https://github.com/amazon-ion/ion-intellij-plugin/releases/latest) and use the [install from disk](https://www.jetbrains.com/help/idea/managing-plugins.html#install_plugin_from_disk) method to add it to IntelliJ.
